	このプロジェクトは、AI を補助として使いながらコードを書き、AI と PLC を組み合わせた仕組みや考え方を試すための個人開発です。AI にすべて任せるのではなく、自分で考えた構成や流れをベースにしつつ、実装や修正の段階で AI を活用しています。完成品を作ることよりも、仕組みや発想を整理しながら試行錯誤することを目的としています。
	这是我的一个个人 C++ 项目，最开始的目的是用ai控制plc，将用户的自然文本转换成c++程序可跑的类容，然后通过网络协议传输控制plc，实际上打算开发ai上位机系统的。

	整体思路是将 PLC、本地程序以及 AI 连接在一起，由 PLC 负责现场数据采集和基础控制，本地程序作为上位机读取 PLC 数据，再将这些数据交给本地运行或通过接口调用的 AI 进行处理，AI 根据数据给出判断或建议，最后再由本地程序把结果写回 PLC。

	程序主要就是定义各种文件格式，然后让ai按照格式输出程序解析，通过通讯输入进plc，目前用了简单的snap7能够控制s7系列的plc1了。

	整个项目使用 C++ 实现（因为只会c++），后续可能会添加其他代码（还得学），结合了一些常用的外部库来完成 PLC 通信、网络请求以及数据解析等功能，实现方式就是先跑起来再优化。

  界面方面暂时不考虑使用 Qt，而是计划后续直接接入 Live2D 模型，把重心放在逻辑验证和交互表现上。

	平时的开发方式也比较随意，一般先在网上查找现有实现或思路，能用就改，找不到就参考以前写过的代码自己搭结构，写完之后再让 AI 帮忙看看逻辑是否合理，最后自己慢慢调试和修正问题。

这个仓库主要用于记录代码和思路，大模型文件和运行库都保存在本地环境中，没有一并上传（太大了而且可替换所有不上传了其实就是上传不了），整体会作为一个长期的个人开发项目持续维护，并随着学习和实践不断调整方向。
